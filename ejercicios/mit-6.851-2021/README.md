# Ejercicios EDA Final - MIT 6.851 Spring 2021

Esta carpeta organiza material de practica para el final de EDA usando como fuente principal el curso publico 6.851 Advanced Data Structures (Spring 2021), de Erik Demaine, y alineandolo con los cuatro bloques indicados para el examen:

1. Integer data structures
2. Static trees, RMQ y LCA
3. Strings
4. Dynamic graph

## Estructura

- `originales/psets/`: PDFs oficiales de problem sets descargados desde MIT.
- `originales/solutions/`: soluciones oficiales publicas disponibles en MIT.
- `temas/`: guias por tema en Markdown, con ruta de estudio y problemas recomendados.
- `latex/eda_final_workbook.tex`: hoja propia de ejercicios y soluciones/esbozos en LaTeX, enfocada en lo que probablemente sirve para el examen a papel.

## Restriccion de uso

No se transcribieron enunciados ni soluciones oficiales largas dentro de los archivos editables. Los PDFs oficiales quedan archivados como referencia local y las hojas `.md/.tex` contienen clasificacion, rutas y soluciones propias/resumidas para estudiar.

## Orden recomendado

1. `temas/strings.md`
2. `temas/static_trees_rmq_lca.md`
3. `temas/dynamic_graph.md`
4. `temas/integer.md`
5. `latex/eda_final_workbook.tex`

Para el examen, practica escribir cada solucion en una pagina: idea, estructura, pseudocodigo, prueba de correctitud y complejidad.
